# Wearable Sensor with AI for Fall Prevention 👵🧠

 Overview
This project focuses on developing a wearable sensor system integrated with Artificial Intelligence to detect and prevent falls in elderly individuals. The system monitors movement patterns and alerts caregivers in case of abnormal activity.

---
 Objectives
- Detect falls in real-time using sensors
- Use AI to predict risky movements
- Send alerts to caregivers or family members
- Improve safety for elderly people

---

 How It Works
1. Sensors collect motion data (accelerometer, gyroscope)
2. Data is processed using AI algorithms
3. System detects abnormal patterns (like sudden falls)
4. Alert is triggered to notify caregivers
